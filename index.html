比较运算符

大多与c语言相同

不同点：

===

绝对等于，如果 x 等于 y，且它们类型相同，则返回true。如5===”5” 返回 false

<>

不等于，如果 x 不等于 y，则返回 true。如5<>8 返回 true

!==

绝对不等于，如果 x 不等于 y，或它们类型不相同，则返回 true。5!==”5” 返回 true
